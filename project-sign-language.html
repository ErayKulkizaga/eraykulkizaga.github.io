AI-Powered Sign Language Translation

Real-time translation to and from sign language with ML/CV.

Female hand by deep3dstudio on Sketchfab

This project aims to build a two-way communication system that bridges the gap between hearing and hearing-impaired individuals through real-time sign language translation.

 

  • Sign Language → Text + Speech : A camera captures sign language gestures. The system detects hand and body movements, converts them into text, and generates speech output.
  • Text → 3D Avatar (Sign Animation) : When a user enters text, a 3D avatar interprets and performs the corresponding sign language gestures.
